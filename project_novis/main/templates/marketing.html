OAuth Provider

Use project novis as a OAuth provider to authenticate users to your ham radio related service. As we are validating the ownership of call sings you can make sure that a user owns a specific call sign. No more call sign validations required on different sites!

View details »

API

We provide an open and modern REST API for all kinds of ham radio applications. As our APIs are based on Swagger/Open-API it can easily be integrated in your application with the help of auto-generated code.

View details »

Hamnet Authentication

At the moment authentification to Hamnet access points is a mess. We try to solve is problem and make the hamnet more secure by restricted the access to access points to our users which validated their call sign. Therefore we provide a RADIUS Server which can be used by your access point with WPA2 Enterprise.

View details »

Open-Source

project novis is free (as beer) and open-source software. We believe in the open ham radio spirit and want to provide an open platform for various tech related ham radio services.

Ham Radio Database
